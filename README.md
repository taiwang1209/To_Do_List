說明
---------------
- To-Do List，記錄待辦事項
- 對資料庫CRUD之操作，包含新增、修改、刪除待辦事項



使用套件
---------------

- [Bootstrap 5](https://getbootstrap.com/docs/5.0/getting-started/introduction/)
- [fontawsome](https://fontawesome.com/)


編寫語言
---------------

- **Flask** - a micro web framework written in Python
- 資料庫使用SQLite


使用說明
---------------

```sh
# 建立資料庫
from app import db
db.create_all()
exit()
# 接著啟動app.py
flask run
```
